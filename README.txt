# OnboardingGame
Task-list Game

This project is meant to be used to create a task/to do list for use at Phoniro. This task list is supose to represent the Onboarding 
process that employees has to partake in. It's this projects intention to create a task list that will make this process more fun by
providing the user a virtual profil within the app. Allowing the user to customize their apperance in the app by using in game currency
given to them by completing task needed to done during the Onboarding process.
